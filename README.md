# Helbert Professional Cleaning Services

Live Website -  https://www.helbertmaintenance.com/

A full-stack professional cleaning services website built to handle real customer inquiries, quote requests, and business contact management with a modern UI and scalable backend.

##  Features

- Responsive and mobile-friendly UI
- Commercial & Residential service pages
- Quote request form with backend integration
- Contact form with real-time email delivery
- SEO optimized pages
- Google Maps integration
- Deployed on production servers

##  Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- React Router
- Axios
- React Helmet (SEO)

### Backend
- Node.js
- Express.js
- MongoDB
- Nodemailer (Email service)
- REST APIs
- Environment Variables (.env)
